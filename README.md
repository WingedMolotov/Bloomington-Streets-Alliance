## Features
- Primer to advocacy - what to advocate for and how to do it
- Action alerts
- Newsletter signup
- Helpful links to other groups and resources
- Mission/vision/goals

## How it works
Made using vanilla HTML, CSS, and JS, relying heavily on flexbox. Some interactive functions were done through JS. Newsletter uses a custom front-end, backed by Buttondown.
